This folder contains upgrades.txt

upgrades.txt contains the data of all upgrades in the base game.

Honest John's has a section that sells the upgrades. It draws from this data.

Data format as follows: (-1 if Not Applicable)
0   , 1              , 2    , 3          , 4     , 5               , 6 ...
name, upgrade game id, price, sprite path, effect, amount increased, duration
